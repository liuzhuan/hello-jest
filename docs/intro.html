开始使用

开始使用

安装

npm install --save-dev jest
yarn add --dev jest

编写业务代码，比如 sum.js：

function sum(a, b) {
    return a + b
}
module.exports = sum

创建测试用例，比如 sum.test.js：

const sum = require('./sum')

test('adds 1 + 2 to equal 3', () => {
    expect(sum(1 + 2)).toBe(3)
})

更新 package.json，增加测试命令：

{
    "scripts": {
        "test": "jest"
    }
}

运行测试用例：

npm test

我们刚刚成功执行了第一个测试用例！

上面测试用例用到了 expect 和 toBe 测试两个数值是否相等。
